- Node.js (v14+ recommended)

### Installation (Windows & Linux)

1. Clone the repository
```bash
git clone https://github.com/yourusername/virtual-talent-hub.git
cd virtual-talent-hub
```

2. Install Dependencies
```bash
npm install
```

3. Run the Website
```bash
npm start
```
